## What Was Fixed (2026-02-24 DSP Rework)

### ✅ 1. Delay Concept — Fixed (ARCHITECTURAL)
- **Old**: grains tracked write head via samplesSinceSpawn, delay effect was relative/implicit
- **New**: `spawnGrainAdvanced` accepts `startReadPos` pre-computed by caller as `(writePos - delaySamples + kBufferSize) & kBufferMask`
- Grains now read from a FIXED SNAPSHOT position: `startReadPos + fractionalPos` (absolute)
- New `readBufferHermiteAbs(ch, absPos)` function for absolute-position reads
- Changing delay_time now clearly shifts which historical moment grains read from

### ✅ 2. Freeze Behavior — Redesigned (smear)
- **Old**: looped a section of audio
- **New**: `freezeSnapPos` updated every sample when NOT frozen (`writePos - delaySamples`)
- When freeze activates, `freezeSnapPos` locks → all new grains spawn at the exact frozen moment
- Granular engine "smears" that single moment over time with overlapping grains
- Mutation + Freeze = grains at the frozen moment degrade over feedback passes

### ✅ 3. Filters — Upgraded to 24dB/oct + Applied to Wet Output
- `TwoPoleFilter` (12dB/oct) replaced by `FourPoleFilter` (24dB/oct, cascaded TwoPoleFilter)
- `feedbackHP` and `feedbackLP`: now FourPoleFilter
- New `wetHP` and `wetLP` (FourPoleFilter): applied to wet signal BEFORE dry/wet mix
- Filters are now immediately audible (not just in the feedback path)

### ✅ 4. Reverse / Mutation / Freq Spread — Amplified
- **Mutation**: grain shrinks to 20% (was 40%), pitch drift ±200 cents (was ±50)
- **Freq Spread**: Low freq grains up to 6× longer (was 3×), High freq grains down to 10% (was 35%)
- **Reverse**: architecture corrected — reversed grains now correctly read backward through the fixed snapshot window

### ✅ 5. Dropdown Overflow — Fixed (UI)
- `.combo-dropdown`: `top: 100%` → `bottom: 100%`, `margin-top: 2px` → `margin-bottom: 2px`
- Box shadow also flipped to upward direction

### ⚠️ 6. Feedback Sustainability — Unchanged (monitoring)
- Feedback path is architecturally correct: `input + fb → write at writePos`
- With fixed-snapshot grains, feedback circulates after delaySamples
- If still inaudible after testing: consider writing feedback directly at `freezeSnapPos` region

### ⚠️ 7. Tempo Sync Bar Boundary — Deferred
- BPM is correctly read from host playhead
- Bar-boundary snap (using PPQ position) not yet implemented
- Deferred to next iteration
